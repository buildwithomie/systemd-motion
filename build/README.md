# Systemd Motion - Quick Install

## Installation

```bash
# Download and extract
wget https://github.com/yourusername/systemd-motion/archive/refs/heads/main.zip
unzip main.zip
cd systemd-motion-main

# Install
bash scripts/simple_install.sh
```

## Usage

```bash
# Check status
motion-ctl status

# Start/stop service
motion-ctl start
motion-ctl stop

# Launch GUI (optional)
motion-gui
```
